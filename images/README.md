# 背景图片说明

请将你选择的半透明背景图片命名为 `background.png`，并放在此文件夹中。

## 推荐的背景图片特性

- **格式**: PNG（支持透明度）
- **尺寸**: 建议 1920×1080 或更大
- **透明度**: 图片本身应该是半透明的，或者通过插件设置调整透明度
- **风格**: 建议选择与流萤主题相符的图片，如：
  - 森林夜景
  - 萤火虫飞舞的场景
  - 梦幻的星空
  - 柔和的绿色渐变

## 如何调整背景透明度

在插件设置面板中，可以调整背景透明度（0-100%）。

## 注意事项

- 背景图片会覆盖整个浏览器窗口
- 背景是固定的，不会随页面滚动
- 背景不会影响页面内容的交互
- 如果不需要背景，可以将透明度设置为 0%
